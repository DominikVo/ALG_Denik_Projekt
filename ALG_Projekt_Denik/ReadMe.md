# Projekt ALG Deníček

## Popis
Úkol pojednával o vytvoření konzolové aplikace deníčku. Hlavní složkou projektu je linked list.
## Funkce
Program dokáže procházet ''.json'' jako deníček.  
Uživatel může přidávat, mazat, upravovat a zobrazovat záznamy v deníčku.
## Struktura projektu
- `Program.cs`: Hlavní soubor aplikace obsahující ovládání konzole a volání metod
- `DenikActions.cs`: Třída obsahující metody pro operace s deníčkem
- `LinkedList.cs`: Implementace linked listu
- `Node.cs`: Jednotlivé části linked listu
- `Data.cs`: Data daného nodu
- `FileHandler.cs`: Třída pro práci se soubory (.json)
## Použití
1. Stáhněte si repozitář.
2. Otevřete projekt v C# vývojovém prostředí (např. Visual Studio).
3. Spusťte aplikaci.
4. Postupujte podle pokynů v konzoli pro správu deníčku.
## Požadavky
- .NET Framework nebo .NET Core
